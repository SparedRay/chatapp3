Prerequisitos

Para iniciar el desarrollo debes tener las siguientes dependencias:

  • Git
  • Node.js (with NPM)
  • Bower
  • Ember CLI
  • PhantomJS

Instalacion

  • git clone git@github.com:SparedRay/chatapp3.git al repositorio
  • cd al nuevo directorio
  • npm install
  • bower install

Running / Development

  • ember server
  • Prueba la aplicacion en http://localhost:4200.

Code Generators

prueba los generadores con ember help generate

Running Tests

  • ember test
  • ember test --server

Building

  • ember build (development)
  • ember build --environment production (production)
